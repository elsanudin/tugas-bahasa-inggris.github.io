TEKNOLOGI REKAYASA PERANGKAT LUNAK

Kode program

kode program adalah serangkain intruksi yang digunakan oleh seorang pemrogram untuk menginstruksikan komputer atau perangkat lunak lainnya. Kode program ini dapat ditemukan dalam berbagai bahasa pemrograman yang ada, seperti C++, Python, dan HTML.

Bahasa pemrograman

Bahasa pemrograman - adalah serangkaian instruksi yang digunakan oleh seorang pemrogram untuk menginstruksikan komputer atau perangkat lunak lainnya.

MVP (Minimum Viable Product)

MVP (Minimum Viable Product) - adalah produk dengan fitur dasar yang memiliki kegunaan tinggi, meskipun bentuknya belum terlalu canggih. MVP dikembangkan untuk menarik perhatian pengguna, memvalidasi ide bisnis, mempelajari pengalaman pengguna, dan menguji potensi keberhasilan produk di pasar secara langsung.

Text editor

Text editor - adalah perangkat lunak yang digunakan oleh para programmer untuk membaca dan menulis bahasa pemrograman. Text editor dilengkapi dengan berbagai fitur, seperti cut and paste, serta kemampuan untuk menyesuaikan tampilan, seperti mengubah skema warna, teks, dan ukuran font.

Algoritma pemrograman

Algoritma pemrograman - adalah serangkaian langkah atau instruksi yang ditetapkan untuk menyelesaikan masalah secara sistematis.

Artificial Intelligence (AI)

Artificial Intelligence (AI) - atau Kecerdasan Buatan adalah bidang dalam ilmu komputer yang berfokus pada pemahaman bahasa alami, pengambilan keputusan, pengenalan pola, dan penyelesaian masalah kompleks.

Bit

Bit - Singkatan dari "binary digit", bit adalah satuan terkecil dalam sistem penyimpanan komputer. Bit dapat berisi nilai 0 atau 1 dan digunakan untuk menyimpan data dalam komputer.

Tag

Tag - adalah penanda awalan dan akhiran dari sebuah elemen di HTML yang dibuat dengan kurung siku ( <...> ).

ISP

ISP - (Internet Service Provider) adalah perusahaan yang menyediakan layanan akses internet kepada pengguna. Mereka menghubungkan pengguna internet ke jaringan global, memfasilitasi akses ke situs web, email, dan layanan online lainnya.

Motherboard

Motherboard - Komponen fisik komputer yang menjadi tempat seluruh komponen lain.

Stored Program Concept

Stored Program Concept - adalah konsep dalam desain komputer di mana instruksi dan data disimpan dalam memori komputer, yang memungkinkan komputer untuk secara otomatis mengeksekusi serangkaian instruksi yang disimpan.

System Control Programs

System Control Programs - Program kontrol sistem. adalah program yang mengendalikan, mengelola, mengordinasi, dan mengelola sumber daya dan fungsi komputer.

Operating System

Operating System - Sistem operasi. merupakan perangkat lunak yang menangani aspek penting dari manajemen sistem, seperti penyimpanan dan akses memori, pengelolaan sumber daya, dan koordinasi antara berbagai komponen sistem.

Multiprocessing

Multiprocessing - adalah konsep dalam komputer yang memungkinkan beberapa unit pengolahan pusat (CPU) dalam satu sistem komputer untuk beroperasi secara bersamaan.

Virtual Memory

Virtual Memory - adalah teknik manajemen memori yang dikembangkan untuk multitasking kernel. Ini memisahkan "memori pengguna" (virtual) dari "memori fisik" dan membuat ruang melintasi (program) dapat lebih besar daripada memori fisik yang ditanpa.

File

File - adalah kumpulan data yang disimpan dalam sebuah unit penyimpanan, seperti hard drive, flash drive, atau media penyimpanan lainnya. File dapat berupa dokumen teks, gambar, video, atau program komputer.

SaaS (Software as a Service)

SaaS (Software as a Service) - Model pengiriman perangkat lunak di mana perangkat lunak dan data terkait dihosting.

User Experience (UX)

User Experience (UX) - Proses merancang produk melalui pendekatan pengguna.

User Interface (UI)

User Interface (UI) - Tampilan visual desain sebuah sistem yang memungkinkan pengguna terhubung.
